# Android Media Scanner by Warranty Voider
<img width="1803" height="719" alt="image" src="https://github.com/user-attachments/assets/9ea339e3-e6fb-4bf4-8e88-3ac0515c2013" />


Tool to quickly find and extract media files from android phones

## Usage
- Attach phone(s)
- Select file transfer as usb option
- Start tool
- Select drives to scan on the left side
- Select scan profile and press scan
- In result tab you can preview the files and select them
- In export tab you can select option to export media files
- Then press "Copy..."
